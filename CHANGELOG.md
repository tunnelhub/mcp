# Changelog

Todas as mudancas relevantes deste projeto serao documentadas neste arquivo.

O formato e baseado em Keep a Changelog e este projeto segue Semantic Versioning.

## [1.0.9] - 2026-03-14

### Adicionado

- GitHub Releases automatizadas disparadas por tags de versao.
- Publicacao no npm usando Trusted Publishing com GitHub Actions OIDC.
- Documentacao do fluxo de release para incrementos patch, minor e major.

### Alterado

- O workflow de release agora atualiza o npm na CI para atender os requisitos do trusted publishing.

## [1.0.8] - 2026-03-14

### Alterado

- Validacao interna do fluxo de release para publicacao no npm e GitHub Releases.

## [1.0.7] - 2026-03-14

### Adicionado

- Automacao inicial do repositorio para tags de release e assets publicados nas releases.
